# Common files repository of system software team in Rebellions

This is the repository for storing the common files used in repositories of system software team

# Directories

## `public`

This branch stores the files used in all repository (UMD, KMD, QEMU, FW)

- E.g., `qman_if_common.h`, etc.

## `private`

This branch stores the files used in the specific repositories (UMD, QEMU, FW)

- E.g., `qman_if.h`, etc.
